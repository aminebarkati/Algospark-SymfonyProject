# Controller Code Breakdown

Each row below covers one controller method, its role, and the Symfony technology it uses.

| Controller            | Method            | Role                                                                                                                               | Symfony technology used                                                                                                                              |
| --------------------- | ----------------- | ---------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| HomeController        | index             | Builds the homepage dashboard with counts, recent submissions, the featured problem, and current-user stats.                       | Route, AbstractController, Doctrine repositories, Twig render, Security token user lookup                                                            |
| FavoriteController    | add               | Adds a favorite user from a JSON payload and rejects invalid or self-favorites.                                                    | Route, IsGranted ROLE_USER, Request JSON decoding, FavoriteRepository, JsonResponse                                                                  |
| FavoriteController    | delete            | Removes a favorite user from a JSON payload.                                                                                       | Route, IsGranted ROLE_USER, Request JSON decoding, FavoriteRepository, JsonResponse                                                                  |
| SubmissionController  | index             | Renders the submissions list page.                                                                                                 | Route, SubmissionRepository, Twig render                                                                                                             |
| SubmissionController  | create            | Validates the submission payload, persists the Submission, writes the source file to storage, and returns a JSON redirect payload. | Route, IsGranted ROLE_USER, Request form and file access, ManagerRegistry, Doctrine ORM, filesystem write, JsonResponse                              |
| SubmissionController  | filter            | Returns filtered submissions as JSON for the submissions list UI.                                                                  | Route, Request query access, SubmissionRepository, JsonResponse                                                                                      |
| SubmissionController  | recent            | Returns the current user’s recent submissions for a problem.                                                                       | Route, IsGranted ROLE_USER, Request query access, SubmissionRepository, JsonResponse                                                                 |
| ProblemController     | index             | Renders the latest problems list.                                                                                                  | Route, ProblemRepository, Twig render                                                                                                                |
| ProblemController     | show              | Loads a problem, its sample test cases, and enabled languages for the detail page.                                                 | Route, ProblemRepository, TestCaseRepository, LanguageRepository, Twig render, not found handling                                                    |
| ProblemController     | new               | Creates a new problem and its test cases for admins.                                                                               | Route, IsGranted ROLE_ADMIN, Request form access, Doctrine EntityManagerInterface, ProblemRepository, TestCase entity persistence, redirect response |
| ProblemController     | renderProblemForm | Renders the problem form with an optional validation error message.                                                                | Twig render helper                                                                                                                                   |
| AuthController        | ajaxLogin         | Authenticates an existing user from posted credentials and starts the Symfony session.                                             | Route, Request form data, ManagerRegistry, UserPasswordHasherInterface, UserAuthenticatorInterface, custom LoginFormAuthenticator, JsonResponse      |
| AuthController        | ajaxRegister      | Creates a new user, hashes the password, and authenticates the new account.                                                        | Route, Request form data, ManagerRegistry, UserPasswordHasherInterface, UserAuthenticatorInterface, custom LoginFormAuthenticator, JsonResponse      |
| AuthController        | logout            | Symfony firewall logout placeholder; the firewall intercepts it before the method body runs.                                       | Route, security firewall logout                                                                                                                      |
| ProfileApiController  | updateAvatar      | Uploads a new avatar image, stores it, and updates the target user record.                                                         | Class-level IsGranted ROLE_USER, Route, Request file access, ManagerRegistry, Doctrine ORM, filesystem move, JsonResponse                            |
| ProfileApiController  | updateProfile     | Updates username, email, and bio with uniqueness checks.                                                                           | Class-level IsGranted ROLE_USER, Route, Request form data, ManagerRegistry, UserRepository, Doctrine ORM, JsonResponse                               |
| ProfileApiController  | updatePassword    | Changes the password, requiring the current password unless an admin edits someone else.                                           | Class-level IsGranted ROLE_USER, Route, Request form data, ManagerRegistry, UserPasswordHasherInterface, Doctrine ORM, JsonResponse                  |
| ProfileApiController  | updateRole        | Toggles admin status for a user.                                                                                                   | Class-level IsGranted ROLE_USER, method-level IsGranted ROLE_ADMIN, Route, Request form data, ManagerRegistry, Doctrine ORM, JsonResponse            |
| ProfileApiController  | deductPoints      | Subtracts rating points from a user.                                                                                               | Class-level IsGranted ROLE_USER, method-level IsGranted ROLE_ADMIN, Route, Request form data, ManagerRegistry, Doctrine ORM, JsonResponse            |
| ProfileApiController  | deleteAccount     | Deletes a user account.                                                                                                            | Class-level IsGranted ROLE_USER, method-level IsGranted ROLE_ADMIN, Route, Request form data, ManagerRegistry, Doctrine ORM, JsonResponse            |
| ProfileApiController  | resolveTargetUser | Resolves the editable user target while enforcing role-based access.                                                               | Request form data, ManagerRegistry, Security isGranted checks                                                                                        |
| ProfileApiController  | serializeUser     | Converts a User entity into a JSON-ready array.                                                                                    | Controller helper used by JsonResponse payloads                                                                                                      |
| UsersController       | index             | Renders the admin-only user list page.                                                                                             | IsGranted ROLE_ADMIN, UserRepository, Twig render                                                                                                    |
| LeaderboardController | index             | Renders users ordered by rating for the leaderboard.                                                                               | Route, UserRepository, Twig render                                                                                                                   |
| ProfileController     | edit              | Renders the profile editor for the current user and optionally lets an admin target another user.                                  | IsGranted ROLE_USER, Route, Request query access, UserRepository, Twig render, Security isGranted                                                    |
| ProfileController     | show              | Renders a public user profile and favorite status.                                                                                 | Route, UserRepository, FavoriteRepository, Twig render, Security isGranted                                                                           |
| ErrorController       | show              | Chooses the appropriate error template for 403, 404, or 500 responses.                                                             | Route, Twig exception templates, Response status handling                                                                                            |
